😀 HTML 😀

HTML utiliza etiquetas de marcas, que consisten en breves instrucciones de comienzo y final mediante las cuales se determina la forma en la que debe aparecer en su navegador de texto, imagenes y los demas elementos.

Toda etiqueta se identifica porque esta encerrada entre los signos menor que (<) y mayor que (>), y algunas tienen atributos que pueden tomar algun valor. En general las etiquetas se aplicaran de dos formas especiales:

Se abren y se cierran, como por ejemplo <b>negrita</b>, que se veia en su navegador web como negrita

No pueden abrirse y cerrarse, como <hr/>, que se veia en su navegador web como una linea horizontal

Todo documento HTML debe contener:

  1. Un elemento html que contiene todo el documento
  2. Una cabecera llamada head
  3. Un cuerpo, llamado body, que el codigo html que se renderiza en el ordenador

Dentro de un documento html puede encontrarse lo siguiente:

  • Texto,Imágenes,Videos
  • Scripts en lenguaje javascript
  • Hojas de estilo CSS
  • Formulario de ingreso de datos
  • Enlaces a archivos externos
  • Y otras caracteristicas detalladas en https://www.w3.org
